│

| Folder/File | Description |
|------------|-------------|
| `LICENSE` | Project license file |
| `README.md` | Main project documentation |
| `data/raw` | Original, immutable dataset |
| `data/processed` | Cleaned and processed dataset |
| `models` | Saved models and predictions |
| `notebooks` | Jupyter notebooks for experiments |
| `reports` | Generated reports and analysis |
| `reports/figures` | Figures and plots used in reports |
| `requirements.txt` | Python dependencies |
| `src` | Source code of the project |



Install all required dependencies using:

pip install -r requirements.txt

The `requirements.txt` file contains all the Python packages needed to run the project.

Example `requirements.txt`:

numpy
pandas
matplotlib
scikit-learn
jupyter
